感覺比較麻煩..
現在我們可以在**Microsoft Azure** 的 **Azure App Service** 上直接 Deploy 存在
**TFS**, **Visual Studio Service**, **GitHub**, **DropBox**, **Bitbucket** 上的 **.Net Core** 程式
