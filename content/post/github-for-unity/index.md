## 使い方
プロジェクトごとに行う必要があります。
1. Asset Storeから「GitHub for Unity」をインポート
2. メニューの`Window`>`GitHub`を選択  
    GitHubタブが開く
3.  **(初回のみ)** `Sign In`をクリック  
    以下のダイアログが表示されるので、GitHubアカウントでサインインする
