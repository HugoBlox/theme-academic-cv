GitHub for Unityの紹介

Juris710
2020/04/30 投稿 ( 2021/03/16 最終更新 )
プロジェクト

使用したUnityのバージョン

Unity 2018.4.22f1

導入方法

プロジェクトごとに行う必要があります。

  1. Asset Storeから「GitHub for Unity」をインポート
  2. メニューでWindow>GitHubを選択
    GitHubタブが開きます。
  3. (初回のみ) Sign Inをクリック
    ダイアログが表示されるので、GitHubアカウントでサインインしてください。
  4. Initialize a git repository for this projectをクリック
    少し待つと、タブの中身が変化します。

使い方

Changesを選択してください。前回保存した状態から変更があったファイルが一覧表示されます。

変更を保存する

  1. リストから保存したい変更をチェックマークをクリックして選択
  2. Commit summaryに変更内容を記述
  3. Commit descriptionに変更内容の詳細を記述（任意）
  4. Commit to [master]をクリック

変更内容を確認する

前回保存した状態と現在の状態との差分を見ることができます。

差分を比較するソフトウェア(WinMergeなど)をインストールする必要があります。

  1. リストから差分を見たいファイルを右クリック
  2. Show Diffをクリック

変更を取り消す

加えた変更を無かったことにして、前回保存した状態に戻すことができます。

  1. リストから変更を取り消したいファイルを右クリック
  2. Discardをクリック

その他

比較ソフトウェアを変更したい

比較ソフトウェアが複数インストールされていて、変更したい場合

  1. UnityのPreferenceを開く
  2. External Toolsを選択
  3. Revision Controll Diff/Mergeでソフトウェアを選択
GitHubやGitに詳しい人向け

Revert

  1. Historyを選択
  2. コミットを右クリック
  3. Revertをクリック
