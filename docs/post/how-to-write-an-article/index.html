はじめに

このサイトでの記事の書き方をまとめました。英語でも構わない場合は公式ドキュメントも参照してください。

Markdown記法

全ての記事はMarkdown記法で書く必要があります。書き方についてはこのサイトを参照してください。

画像を挿入する場合

![代替テキスト](./画像名)

「画像名」には拡張子も含めてください
画像のサイズを変えたい、または画像を左寄せ・右寄せしたい場合は下の画像の挿入を参照してください

Shortcodes

ShortcodesとはHugo独自の機能で、Markdownファイルに書くことができるスニペットです。

画像の挿入

{{<figure src="./画像名" title="代替テキスト">}}
「画像名」には拡張子も含めてください

画像の左寄せ、右寄せ

左寄せ 

{{<figure src="./画像名" title="代替テキスト"  class="left">}}

右寄せ 

{{<figure src="./画像名" title="代替テキスト"  class="right">}}

左寄せ(float) 

{{<figure src="./画像名" title="代替テキスト"  class="float-left">}}

右寄せ(float) 

{{<figure src="./画像名" title="代替テキスト"  class="float-right">}}

他にも、画像の幅・高さなどをカスタムすることができます。詳しくは公式ドキュメント を参照してください。

Youtube動画埋め込み

{{<youtube 動画のID>}}
YoutubeのURLは「https://www.youtube.com/watch?v=動画のID」という形式になっています

Unity WebGLの埋め込み

ランダムさんの記事の内容を少し改変した自作Shortcodeです。Unity WebGLのゲームをページ内でプレイすることができます。

{{<unity src="URL">}}

ツイートの埋め込み

{{<tweet ツイートID>}}
ツイートのURLは「https://twitter.com/…/status/ツイートID」という形式になっています。

折りたたみ

{{<spoiler "折りたたまれている時に表示する文字列">}}
  展開された時に表示する文章(複数行可)
{{</spoiler>}}

折りたたまれている時、展開された時ともに、Markdown記法を用いた文章が使えます。例えば

{{<spoiler "`Hello`">}}
  foo
  *bar*
  > Hello, World!
{{</spoiler>}}

のように書けば、下のように表示されます。

Hello

foo
bar

Hello, World!

ページへのリンク

{{<cite page="表示したいページのパス(ホームからの相対パス)" view="表示形式">}}

表示形式には1から4の数値が指定できます。

例えば、{{<cite page="project/opu-virtual-festival-2020" view="1">}}と記述すると、下のように表示されます。

府大バーチャル学祭
Juris710, Nobu, ヤマグチ

view="2"に変えると、下のように表示されます。

府大バーチャル学祭
OPU Virtual Festival
Juris710, Nobu, ヤマグチ

view="3"に変えると、下のように表示されます。

Juris710, Nobu, ヤマグチ
2020/04/28 投稿 ( 2021/03/16 最終更新 )
府大バーチャル学祭

OPU Virtual Festival

view="4"に変えると、下のように表示されます。

Juris710, Nobu, ヤマグチ (2020). 府大バーチャル学祭.

その他

数式

$\LaTeX$ 記法を用いることで数式を表示することができます。数式は$$で囲む必要があります。

$$ F=G\frac{Mm}{r^2} $$

下のように表示されます。 $$ F=G\frac{Mm}{r^2} $$ $$の代わりに$で囲むと $ F=G\frac{Mm}{r^2} $ のようにインライン表示できます。

Diagrams

mermaid記法を使った図の挿入が可能です。

```mermaid
graph TD;
  A-->B;
  A-->C;
  B-->D;
  C-->D;
```

以下のように表示されます。

graph TD;
  A-->B;
  A-->C;
  B-->D;
  C-->D;

スライド

Markdown記法で簡易的なスライドを作成することができます。

{{<slide-frame src="スライド名">}}
